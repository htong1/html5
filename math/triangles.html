Triangle Evaluator :D

Design an HTML5 app that evaluates triangles. Use 3 input boxes to let the user enter the three sides by length. Validate that the input values are between 1 and 300. Do not restrict the order of the values (e.g., the longest first) Validate that the lengths entered form a triangle; Otherwise, alert user to change the inputs, rejecting the inputs. If the input lengths form a triangle, draw the triangle with the following instruction: https://www.w3schools.com/graphics/svg_polygon.asp Use big letters to inform the user: That is a "SCALENE" or "EQUILATERAL" or "ISOSCELES" or "RIGHT" triangle. It has an area of ___, a base of ___ and a height of ___.

Input the length of sides of your triangle.
